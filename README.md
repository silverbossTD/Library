# Library
Try my library book website at https://silverlibrary.web.app

Download code

## Backend
after download nodejs and npm run:

```
npm install
```
And 
```
npm start
```

## Frontend
after download vue run:

```
npm install
```

and 
```
npm run serve
```

Finally try website at http://localhost:8080/
